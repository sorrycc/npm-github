# npm-github

Open url in browser by npm pkg name.

[![NPM version](https://badge.fury.io/js/npm-github.png)](http://badge.fury.io/js/npm-github)

---

## Install

```bash
$ npm i npm-github -g
```

## Usage

```bash
$ npm-github npm-github
```
